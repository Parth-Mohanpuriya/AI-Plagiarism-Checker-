# AI Plagiarism Checker

AI-powered academic integrity system that analyzes assignment submissions using automated workflows.

## Features
- Google Form submission pipeline
- Google Sheets dashboard
- AI-generated content detection
- Similarity scoring
- Confidence scoring
- Verdict classification
- Ethics & bias review
- Automated plagiarism report generation

## Tech Stack
- Google Forms
- Google Sheets
- n8n
- Groq LLM
- Canva
- JSON Workflow Automation

## Workflow
Student Submission → Google Forms → Google Sheets → n8n → AI Analysis → Results Dashboard → PDF Report

## Files Included
- n8n workflow export
- Final plagiarism report
- Ethics documentation
- Screenshots

## Status
Fully tested with 30+ submissions.
